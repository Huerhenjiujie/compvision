Readme for 2_convOMETIFF

Description:
- Converts the stitched image format from .tif to .ome.tif
- Copies the metadata from .lsm files (one of the tiles in the stitched image) to metadata of stitched image.

Usage:
- Complied jar provided
- Execute with command: ImageConverter.jar <source: .lsm filename> <source: .tif filename> <destination: .ome.tif filename>


Notes:
- Source code provided as a java project.
